# 開発環境の初期化

## npmからパッケージをインストールします

```bash
npm i
```

## ビルドします

```bash
npm run build
```

## Chromeで拡張機能のウィンドウへ移動します

「Manage Extensions」を選択します

![ext](docs/ext-window.png)

## Chromeで「build」のフォルダーを拡張機能として追加しましょう

「Load unpacked extension」を選択します

![Add ext](docs/add-ext.png)

# 開発環境の使い方

自動的に変化がChromeに反映されます

```bash
npm run watch
```
